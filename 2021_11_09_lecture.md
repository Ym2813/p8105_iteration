## Define Function

``` r
mean_and_sd = function(x) {
  
  if (!is.numeric(x)) {
    stop("Argument x should be numeric")
  } else if (length(x) == 1) {
    stop("Cannot be computed for length 1 vectors")
  }
  
  mean_x = mean(x)
  sd_x = sd(x)

  output_df = 
    tibble(mean = mean_x, 
       sd = sd_x)
  # or use list(mean = mean_x, sd = sd_x)
  return(output_df)
}
```
